AI 기반 기술 논문 상담 서비스

일상어로 기술 아이디어를 입력하면, 하이퍼클로바X와 논문 DB를 연동하여 전문적인 기술 인사이트와 구현 가이드를 제공하는 서비스입니다.  

프로젝트 소개
기존 키워드 매칭 방식의 한계를 극복하고, 사용자가 입력한 아이디어의 맥락을 AI가 파악하여 관련 논문을 찾아 분석해 주는 지능형 서비스입니다. 
2일이라는 짧은 기간 내에 최적의 효율을 내기 위해 복잡한 머신러닝 학습 없이 API와 데이터베이스를 유기적으로 조립하여 구축하였습니다.  

주요 기능
지능형 키워드 추출: 하이퍼클로바X를 활용해 사용자 아이디어에서 핵심 기술 키워드를 자동 추출합니다.  
지능형 논문 검색: 추출된 키워드를 기반으로 논문 DB(CSV)에서 관련 연구 자료를 탐색합니다.  
기술 분석 및 상담: 검색된 논문 정보를 바탕으로 하이퍼클로바X가 아이디어에 대한 심층 분석 및 기술적 인사이트를 제시합니다.  


기술 스택
Language: Python   
AI Engine: HyperCLOVA X API   
Database: postgre DB   
Web Framework: Streamlit   
Infrastructure: Naver Cloud Platform (NCP)   

👥 팀 역할 분담
인프라 팀원: NCP 서버 세팅 및 깃허브 관리   
데이터 팀원: 논문 데이터셋(30건) 수집 및 CSV 정제   
개발 팀원 1: 하이퍼클로바X API 연동 함수 구현   
개발 팀원 2: CSV 기반 키워드 검색 로직 구현   
PM/발표 팀원: Streamlit 화면 구현 및 시연 시나리오 작성
